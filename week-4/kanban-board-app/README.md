# React Kanban Board

## Features
- React + Vite setup
- 6+ reusable components
- Todo / In Progress / Done columns
- Add, edit, delete tasks
- useState and useEffect
- LocalStorage persistence
- Drag and drop support
- React Router with 2 pages
- Responsive design

## Run Locally

npm install
npm run dev

## Deploy
- Push to GitHub
- Import project into Vercel or Netlify
- Deploy
